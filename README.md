# EVM File System

Ethereum Virtual Machine network file system.

```bash
  evmfs \
    -h
```

Links pointing to the file system resources are structured in the following way:

```
evmfs://<evm_network_id>/<evmfs_contract_address>/<user_namespace>/<file_hash>
```
